# How to Play
### On Keyboard: 
Left/A and Right/D for player movement

Up/W for shoot

Space for Start and Next Round

### On Mobile:
Arrow Buttons

Good Luck!

# Installation

###To fork my repo:

you will see a button in the UPPER RIGHT hand corner that says Fork. 

### To clone my repo:

git clone https://github.com/ctsmit/EarthDefense.git
